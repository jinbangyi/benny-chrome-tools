Chrome Extension Test Page

Test Instructions:

  1. Install the Chrome extension
  2. Open the extension popup
  3. Set endpoint to: http://localhost:3000/api/metrics
  4. Choose a predefined function or paste custom code
  5. Click "Start Watching"
  6. Click the test buttons below to generate requests

Predefined Functions:

Birdeye Metrics (NEW): Use this code:

function index(response) {
  return 'BUILDIN_birdeye-metrics';
}

Simple Log: Use this code:

function index(response) {
  return 'BUILDIN_simple-log';
}

Count Records: Use this code:

function index(response) {
  return 'BUILDIN_count-records';
}

Legacy Support: Old function names still work:

function index(response) {
  return 'monthly-sum'; // Will use BUILDIN_birdeye-metrics
}
